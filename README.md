# newbie-installer

## Dependencies

### Debian / Ubuntu / Mint
```Shell
$ sudo apt-get update
$ sudo apt-get -y install build-essential
```

### CentOS / RHEL 7/6/5
```Shell
$ sudo yum update
$ sudo yum groupinstall "Development Tools"
```

### Fedora 28/29/30
```Shell
$ sudo dnf update
$ sudo dnf groupinstall "Development Tools"
```
